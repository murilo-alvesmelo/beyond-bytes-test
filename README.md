
# Processo Seletivo Beyond the Bytes

## Rodando localmente

Clone o projeto


Instale as dependências

```bash
  $ npm install
```

Inicie a aplicação

```bash
  $ npm run dev
```

## Hospedagem

https://beyond-bytes-test.vercel.app/

## Dependências utilizadas

- axios
- moment
- react-router-dom
- react-input-mask
- react-modal
- sweetalert2
